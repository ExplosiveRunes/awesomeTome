\textbf{Hit Die:} d8.

\textbf{Class Skills}

The warrior's class skills (and the key ability for each skill) are \linkskill{Climb} (Str), 
\linkskill{Handle Animal} (Cha), \linkskill{Intimidate} (Cha), \linkskill{Jump} (Str), \linkskill{Ride} (Dex), and \linkskill{Swim} (Str).

\textbf{Skill Points at 1st Level:} (2 + Int modifier) x 4.

\textbf{Skill Points at Each Additional Level:} 2 + Int modifier.
